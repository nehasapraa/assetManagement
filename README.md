## Build Setup

``` bash
# install dependencies
yarn
knex migrate:latest
knex seed:run

For Running Application

For Frontend Server
# serve with hot reload at localhost:8080
npm run dev

For Backend Api
npm start
api running at http://127.0.0.1:3300

# run all tests
npm test
```
